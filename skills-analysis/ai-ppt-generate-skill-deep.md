## 📋 概览

AI PPT Generate 是一个基于百度千帆平台的智能 PPT 生成工具。它可以根据用户提供的主题、模板和资源文件，自动生成专业的 PowerPoint 演示文稿。

### 核心价值

- 🎯 **智能生成**: 基于主题自动生成 PPT 内容和结构
- 🎨 **模板支持**: 提供多种内置模板和自定义模板选项
- 📚 **资源整合**: 支持上传文档、图片等资源辅助生成
- 🌐 **多语言**: 支持中英文 PPT 生成
- 📊 **专业模式**: 提供简约和专业两种布局模式

## 🔧 技术架构

### API 端点

所有 API 都基于百度千帆平台：
- **Base URL**: `https://qianfan.baidubce.com/v2/tools/ai_ppt/`

### 三个核心 API

#### 1. PPTThemeQuery - 主题查询
- **路径**: `/get_ppt_theme`
- **方法**: POST
- **功能**: 查询内置的 PPT 主题和模板列表
- **认证**: Bearer Token (BAIDU_API_KEY)
- **返回**: 主题列表（包含 style_id 和 tpl_id）

#### 2. PPTOutlineGenerate - 大纲生成
- **路径**: `/generate_outline`
- **方法**: POST (SSE 流式返回)
- **功能**: 根据主题生成 PPT 大纲
- **关键参数**:
  - `query`: PPT 主题或标题（必需）
  - `resource_url`: 资源文件 URL（可选）
  - `page_range`: 页数范围（1-10, 11-20, 21-30, 31-40, 40+）
  - `layout`: 布局模式（1: 简约, 2: 专业）
  - `language_option`: 语言（zh, en）
  - `gen_mode`: 生成模式（1: 智能润色, 2: 严格遵循）
- **返回**: query_id, chat_id, outline (markdown 格式)

#### 3. PPTGenerate - PPT 生成
- **路径**: `/generate_ppt_by_outline`
- **方法**: POST (SSE 流式返回)
- **功能**: 根据大纲生成最终 PPT 文件
- **关键参数**:
  - `query_id`, `chat_id`: 来自大纲生成 API
  - `outline`: Markdown 格式的大纲（可编辑）
  - `query`: 原始查询
  - `title`: PPT 标题
  - `style_id`, `tpl_id`: 来自主题查询 API
  - `custom_tpl_url`: 自定义模板 URL
  - `ai_info`: 是否在末页显示 AI 生成信息
- **返回**: PPT 文件下载地址

## 💻 实现细节

### 依赖项

- **Python 3.x**
- **requests**: HTTP 请求库
- **环境变量**: `BAIDU_API_KEY`

### 脚本结构

```
ai-ppt-generate/
├── SKILL.md                    # Skill 文档
├── _meta.json                  # 元数据
├── scripts/
│   ├── ppt_theme_list.py       # 查询主题列表
│   ├── ppt_outline_generate.py # 生成大纲
│   └── ppt_generate.py         # 生成 PPT
└── .clawhub/                   # ClawHub 配置
```

### 关键技术点

1. **SSE 流式响应**: 大纲生成和 PPT 生成都使用 Server-Sent Events
2. **Markdown 大纲**: 大纲使用 Markdown 格式，用户可编辑
3. **多阶段依赖**: 后续步骤依赖前序步骤的返回值
4. **错误处理**: 检查 `errno` 字段判断 API 是否成功

## ⚠️ 注意事项

### 依赖关系

- **必须按顺序执行**: 主题查询 → 大纲生成 → PPT 生成
- **参数传递**: 后续步骤需要前序步骤的返回值
- **大纲可编辑**: 用户可以在生成 PPT 前修改大纲内容

### 限制

- **API Key 必需**: 需要百度千帆平台的有效 API Key
- **资源文件格式**: 支持 doc, pdf, ppt, pptx, png, jpeg, jpg
- **自定义模板**: 必须是可下载的 URL
- **SSE 超时**: 流式响应可能需要较长时间，注意超时设置
